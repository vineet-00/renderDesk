Issue Tracking App

Issue management

  • Next.js
  • TypeScript
  • PostgreSQL
  • TailwindCSS

A focused issue-tracking app (in the spirit of Linear) implementing authentication and the core issue workflows: create, view, edit and delete issues. Built with Next.js + TypeScript, styled with TailwindCSS and persisted to PostgreSQL (Instagres). The UI is lightweight and the API is minimal and secure — designed for clear, fast issue management.

Key features: user authentication, issue creation with title/description/labels/assignee, inline editing, soft/hard delete options, and an issues list with simple filters. The app emphasises straightforward UX and a small, auditable server surface for easy maintenance.
